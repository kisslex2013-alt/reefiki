# REEFIKI — Аудит: Accessibility и UX для нетехнических пользователей

> Дата: 4 июня 2026. Часть комплексного аудита незатронутых тем.

## 5.1 Текущий портрет пользователя в аудитах

Все предыдущие аудиты предполагают пользователя, который знает git, Python, CLI, понимает термины frontmatter, YAML, FTS5. Реальная целевая аудитория шире: исследователи, журналисты, менеджеры, студенты.

## 5.2 Барьеры входа

**Барьер 1 — Установка:**
```
Нужно: Python 3.11+, git, pre-commit, опционально uv
Время: 30–90 минут для нетехнического пользователя
Риск отказа: высокий на этапе "Add Python to PATH"
```

**Барьер 2 — Концептуальная модель:**
```
Пользователь должен понять:
- Что такое «проект» в REEFIKI
- Разницу между inbox/, wiki/, seen/, raw/
- Что такое frontmatter и зачем он нужен
- Почему нельзя просто редактировать файлы в Obsidian
```

**Барьер 3 — Язык интерфейса:**

AGENTS.md содержит инструкцию «говори бытовым языком», но:
- Сообщения об ошибках (`FileNotFoundError`, `OperationalError`) — технические
- Поле `useful_when` — не переведено в выводе CLI
- `quarantine_until`, `frontmatter`, `FTS5` — нигде не переводятся

**Барьер 4 — Отсутствие GUI:**
```
Нет: веб-интерфейса, десктопного приложения, мобильного приложения
Есть: CLI + Obsidian как viewer
```

## 5.4 UX-аудит ключевых сценариев

**Первый запуск без IDE:**
```bash
$ python scripts/reefiki.py status
# Текущий вывод:
Project: _template
Inbox: 0 items
# Проблема: пользователь не знает что делать дальше
# Нужно: «Вики пустая. Начни с: python scripts/reefiki.py save <URL>»
```

**Ошибка при отсутствии git:**
```bash
# Текущий вывод:
FileNotFoundError: [Errno 2] No such file or directory: 'git'
# Нужно:
# «Для этой команды нужен git. Установи: https://git-scm.com»
```

**Поиск без результатов:**
```bash
$ python scripts/reefiki.py search "машинное обучение"
# Текущий вывод: (пусто)
# Нужно:
# «Ничего не найдено. Попробуй: 'машин' или 'обучен' (поиск по части слова)»
```

## 5.5 Рекомендации

```python
# 1. Дружелюбные сообщения об ошибках
def user_friendly_exit(user_msg: str) -> None:
    print(f"⚠️  {user_msg}", file=sys.stderr)
    sys.exit(1)

# 2. Подсказки при пустых результатах
if not results:
    print(f"Ничего не найдено по запросу '{query}'.")
    print("Попробуй более короткое слово или проверь /status.")

# 3. Прогресс при долгих операциях
print(f"Индексирую {len(pages)} страниц...", end="", flush=True)
# ... операция ...
print(f" готово ({elapsed:.1f}s)")
```
